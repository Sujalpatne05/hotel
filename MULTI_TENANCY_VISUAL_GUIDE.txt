                         DATA FLOW EXAMPLE
================================================================================

SCENARIO: Dosti Admin logs in and requests menu

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  1. Admin logs in                                                       │
│     Email: admin@dosti.com                                             │
│     Password: ****                                                      │
│                                                                         │
│  2. Backend finds user in database                                      │
│     Query: SELECT * FROM users WHERE email = 'admin@dosti.com'        │
│     Result: User found with restaurant_id = 1                          │
│                                                                         │
│  3. Backend creates session token                                       │
│     Token contains: user_id=1, restaurant_id=1, role=admin             │
│                                                                         │
│  4. Admin requests menu                                                │
│     Frontend: GET /menu                                                │
│     Token sent: restaurant_id=1                                        │
│                                                                         │
│  5. Backend filters menu by restaurant_id                              │
│     Query: SELECT * FROM menu WHERE restaurant_id = 1                  │
│     Result: [Biryani, Butter Chicken]                                  │
│                                                                         │
│  6. Admin sees ONLY Dosti's menu ✅                                     │
│     ├─ Biryani (250)                                                   │
│     └─ Butter Chicken (300)                                            │
│                                                                         │
│  7. Admin CANNOT see other restaurants' menu ✅                         │
│     ├─ Burger (ABC) - NOT VISIBLE                                      │
│     ├─ Pizza (ABC) - NOT VISIBLE                                       │
│     ├─ Samosa (Taj) - NOT VISIBLE                                      │
│     └─ Naan (Taj) - NOT VISIBLE                                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
