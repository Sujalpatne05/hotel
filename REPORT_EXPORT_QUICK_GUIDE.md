# Report Export - Quick Guide

## What Was Fixed
Reports were downloading with corrupted text. Now they export cleanly in proper format.

## How to Export Reports

### From Reports Page
1. Go to **Reports & Analytics**
2. Click **Download PDF** or **Download CSV**
3. File downloads automatically
4. Open with your preferred application

### From Daily Tally Page
1. Go to **Daily Tally Report**
2. Select period (Daily/Weekly/Monthly)
3. Click **Export PDF** or **Export CSV**
4. File downloads automatically

## Export Formats

### PDF Format
- Professional appearance
- Easy to print
- Includes all data
- Proper formatting

### CSV Format
- Opens in Excel/Google Sheets
- Easy to analyze
- Sortable and filterable
- Good for data processing

## File Names
- Reports: `restaurant-report-YYYY-MM-DD.pdf/csv`
- Tally: `daily-tally-YYYY-MM-DD.pdf/csv`

## Troubleshooting

### PDF won't open
- Try different PDF reader
- Check if file downloaded completely
- Refresh page and try again

### CSV shows wrong format
- Open with Excel or Google Sheets
- Check encoding is UTF-8
- Verify comma separation

### Missing data
- Refresh page
- Check if orders are loaded
- Verify data exists
