## 🎯 Common Tasks

### Add Menu Item
1. Login as Admin
2. Menu Management → Add New Item
3. Enter name, price, upload image
4. Save

### Create Order
1. Orders → New Order
2. Select items and quantity
3. Create

### Make Reservation
1. Reservations → New Reservation
2. Enter customer details
3. Select date, time, table
4. Save

### Create Restaurant (Super Admin)
1. Restaurants → Add Restaurant
2. Enter details
3. Admin auto-created
4. Subscription auto-created
